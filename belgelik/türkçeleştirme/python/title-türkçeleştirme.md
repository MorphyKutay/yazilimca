# title() Kullanımı

**AÇIKLAMA**
`title()` yöntemi, her kelimenin ilk harfi büyük olacak şekilde bir dize döndürür.

**GİRDİ**
```
text = 'He's an engineer, isn't he?'
print(text.title())

text = '234 k3l2 *43 fun'
print(text.title())
```
**ÇIKTI**
```
He'S An Engineer, Isn'T He?
234 K3L2 *43 Fun
```
