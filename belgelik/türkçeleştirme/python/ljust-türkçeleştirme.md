# ljust() Kullanımı

**AÇIKLAMA**
`ljust()` yöntemi, belirli bir minimum genişlikte sola yaslanmış bir dize döndürür.

**GİRDİ**
```
string = 'cat'
width = 5

print(string.ljust(width))
```
**ÇIKTI**
```
cat  
```
