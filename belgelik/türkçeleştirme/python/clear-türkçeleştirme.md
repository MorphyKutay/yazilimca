# clear() Kullanımı

**AÇIKLAMA**
clear() yöntemi listedeki tüm öğeleri kaldırır.

**GİRDİ**
```
# Listeyi tanımlamak
list = [{1, 2}, ('a'), ['1.1', '2.2']]

# Listeyi temizlemek
list.clear()

print('Liste:', list)
```
**ÇIKTI**
```
Liste: []
```
