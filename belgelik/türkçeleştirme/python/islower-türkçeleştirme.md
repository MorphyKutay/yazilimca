# islower() Kullanımı

**AÇIKLAMA**
`islower()` yöntemi, bir dizedeki tüm alfabeler küçük harfli alfabeler ise True döndürür. Dize en az bir büyük harf içeriyorsa, False döndürür.

**GİRDİ**
```
s = 'this is good'
print(s.islower())

s = 'th!s is a1so g00d'
print(s.islower())

s = 'this is Not good'
print(s.islower())
```
**ÇIKTI**
```
True
True
False
```
