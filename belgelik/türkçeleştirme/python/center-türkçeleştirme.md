# "center" Sözcüğünün Kökeni

# center() Kullanımı

**AÇIKLAMA**
`center()` yöntemi,dolgu karakterini(boşluk varsayılandır) kullanarak dizeyi ortaya hizalar.

**GİRDİ**
```
txt = "karpuz"

x = txt.center(20)

print(x)
```
**ÇIKTI**
```
          karpuz
```
