# exec() Kullanımı

**AÇIKLAMA**
`exec()` yöntemi, bir dize veya kod nesnesi olan dinamik olarak oluşturulmuş programı yürütür.

**GİRDİ**
```
program = input('Enter a program:')
exec(program)
```
