# bin() Kullanımı

**AÇIKLAMA**
`bin()` yöntemi,belirli bir tamsayının ikili eşdeğer dizesini dönüştürür ve döndürür.

**GİRDİ**
```
sayı = 5
print('5'in ikili eşdeğeri:', bin(sayı))
```
**ÇIKTI**
```
5'in ikili eşdeğeri: 0b101
```
