# continue Kullanımı

**AÇIKLAMA**
`continue` ifadesi, yalnızca geçerli yineleme için bir döngü içindeki kodun geri kalanını atlamak için kullanılır. Döngü sona ermez ancak bir sonraki yinelemeyle devam ettirir.

**GİRDİ**
```
for val in "string":
    if val == "i":
        continue
    print(val)

print("The end")
```
